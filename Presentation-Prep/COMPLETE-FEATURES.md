# EZTeach — Complete Feature List

> Everything included in the EZTeach platform, organized by category.

---

## AI-Powered Features

| Feature | Description |
|---------|-------------|
| AI Lesson Plan Generator | Complete standards-aligned lesson plans generated from subject, grade, topic, and learning style. Includes Bloom's Taxonomy objectives (5 levels), 4 activity types, differentiation for 5 learner levels, CCSS/NGSS standards, assessments, and homework. |
| AI Study Plans | Personalized study plans for students based on performance data and learning style |
| Improvement Tracking | AI monitors student progress in assignments, grades, and game scores with automatic notifications |
| Adaptive Learning | AI gets smarter with each use, learning patterns from your school's data |
