# AppStore-Ready Folder

Everything you need to publish EZTeach on the Apple App Store.

## Contents

| File | What It Is |
|------|-----------|
| `App-Store-Listing.md` | Complete App Store listing — name, subtitle, description, keywords, release notes. Copy-paste ready. |
| `Screenshot-Captions.md` | Captions for each screenshot + required device sizes |
| `App-Store-Connect-Checklist.md` | Step-by-step checklist for uploading to App Store Connect |
| `screenshot-*.png` | App screenshots from iPhone 17 Pro Max simulator |

## Screenshots Included

1. `screenshot-1-student-home.png` — Student home screen (Welcome, grade, school)
2. `screenshot-2-games-hub.png` — EZLearning Games hub (Math, Science, Reading, etc.)
3. `screenshot-3-talker-board.png` — AAC Talker Board (200+ emoji cards)
4. `screenshot-4-free-books.png` — Free Library (Project Gutenberg, Read Together)
5. `screenshot-5-interactive-book.png` — Interactive picture book (Tap the Magic Box)
6. `screenshot-6-picture-books.png` — Readings by Level (Fiction, Non-Fiction, Sci-Fi)

## Quick Start

1. Open `App-Store-Connect-Checklist.md` and follow each step
2. Copy text from `App-Store-Listing.md` into App Store Connect fields
3. Upload screenshots with captions from `Screenshot-Captions.md`
4. Archive and upload from Xcode
5. Submit for review
