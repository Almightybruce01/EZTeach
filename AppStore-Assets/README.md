## App Store Screenshots
Apple requires **real app screenshots** from the iOS Simulator. Use:

1. Xcode → Run on **iPhone 15 Pro Max** simulator
2. Sign in with test account
3. Navigate to: Home, Grades, Sub Requests, Calendar, School Info, Account
4. **Cmd + S** to save each screenshot (saves to Desktop)
5. Upload to App Store Connect → Screenshots → 6.5" Display

Required size: **1284 × 2778 px** (or 1242 × 2688 px)
